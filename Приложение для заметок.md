# Основной интерфейс

- Главная страница со списком заметок (карточки или таблица с заголовком, превью текста, датой создания/изменения).
- Форма создания новой заметки: поля для заголовка, текста, опционально тегов или категории.
- Кнопки действий для каждой заметки: редактировать, удалить, пометить как важную.

## Функциональные возможности

- Добавление заметки: быстрая форма (с горячими клавишами, например Ctrl+N), автоматическое сохранение черновика.
- Удаление: подтверждение удаления, корзина для восстановления.
- Хранение данных: строго на в базе данных, за возможным исключением для корзины для восстановления и для черновиков, которые можно хранить у пользователя.

## Дополнительные фичи

- Поиск по тексту и заголовкам (фильтрация в реальном времени).
- Сортировка: по дате, алфавиту, важности.
- Темная/светлая тема.
- Экспорт в TXT/JSON/PDF.
- Появление кнопок взаимодействия с заметками только при наведении курсора или нажатии на неё.
